Mecánica





Magnitud:

Es la característica de un objeto, sustancia o fenómeno físico que se puede definir de forma numérica.

Magnitud Fundamental:

Una magnitud fundamental no se puede definir en función de ninguna otra. En la mecánica solo se utiliza la longitud, la masa y el tiempo.

Magnitud Derivada:

Una magnitud derivada esta definida por una magnitud fundamental.


Ecuaciones de Dimensiones:

Es el nombre de la expresión matemática que relaciona una magnitud derivada con las fundamentales.


Medida de las Magnitudes:

Medir una magnitud es comparar una cantidad de esa magnitud con otra cantidad, La unidad de una magnitud es la cantidad elegida aleatoriamente que se utilizara como elemento de comparación. Para medir magnitudes se usan instrumentos calibrados: un cronometro, un termómetro, una báscula, etc.


Unidades en el Sistema Internacional:


Concepto de Unidad:

La unidad de una magnitud es la cantidad elegida aleatoriamente que se utilizara como elemento de comparación. Se expresa con un número acompañado por su unidad.

Sistema Internacional de Unidades:

En la antigüedad cada pueblo tenía sus propias unidades. En 1790 se propuso un sistema único llamado “Sistema métrico decimal”. En 1960 se redefinieron las unidades de las magnitudes fundamentales.


Órdenes de Magnitud:

El orden de magnitud de un número es el valor al que se eleva la base 10 cuando se expresa un número en notación científica.


Notación Científica:

Notación científica es la expresión del valor numérico de una sola cifra seguida de una potencia de 10.


Cifras significativas:

El conjunto de dígitos exactos mas los dígitos sometidos a error, de una medida, son las cifras significativas. Los ceros solo cuentan si están entre otras cifras. Existen reglas para el redondeo:

- Cuando el digito descartado es superior a 5, al digito anterior se le suma 1.

- Cuando el digito descartado es menor que 5, el anterior no cambia.

- Cuando el digito descartado es si es 5, al anterior se le suma 1 si es impar, si es par o 0, se queda igual.

- El redondeo en sumas o restas de medidas se hará coincidiendo los decimales del resultado con los del dato que tenga menos cifras decimales.

- Multiplicando y dividiendo se redondea haciendo coincidir el numero de dígitos significativos del que menos tenga.
